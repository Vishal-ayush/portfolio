# Vishal Ayush — Personal Portfolio

A sleek, modern, and highly interactive personal portfolio website tailored for a Cybersecurity Enthusiast and Computer Science Engineering student.

## 🚀 Features

- **Cybersecurity Aesthetic**: A hacker-themed dark mode with neon accents, custom scanline overlays, and an animated terminal interface.
- **Interactive Hero Sections**: 
  - A 3D interactive globe visualizing live global network packet transfers.
  - Background neural network particle animations powered by the HTML Canvas API.
- **Network Scan Animation**: Visually simulated network security scans tracking firewall statuses, phishing detection, and open ports.
- **Fully Responsive Design**: Ensures a seamless viewing experience across desktops, tablets, and smartphones using CSS Grid and Flexbox.
- **Integrated Portfolio Sections**:
  - **Technical Arsenal**: Highlighting programming languages, security frameworks, and tools.
  - **Training & Education**: A timeline of academic background, rigorous DSA training, and certifications.
  - **Build Log (Projects)**: Detailed project cards like *PhishDetect* and *SecureAuth* with GitHub integration.
  - **Impact & Achievements**: Metrics on coding problems solved, community development, and club leadership.

## 🛠️ Tech Stack

This project is built purely with web standards to ensure zero dependencies and blazing-fast loading speeds:

- **HTML5**: For semantic structure and accessibility.
- **CSS3**: Core styling including advanced `@keyframes` animations, CSS Grid/Flexbox layouts, and completely custom CSS Variables.
- **Vanilla JavaScript**: Handles complex Canvas API rendering (Globe, Networks), Intersection Observers for scroll-reveals, and custom typewriter effects.

## ⚙️ Getting Started

Because this portfolio uses a single-file architecture with no build dependencies, setting it up is incredibly simple:

1. Clone or download this repository to your local machine.
2. Open `index.html` directly in any modern web browser (Google Chrome, Firefox, Safari).

*Alternatively, deploy it instantly using GitHub Pages, Netlify, or Vercel by uploading the files.*
